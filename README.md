# 🚀 Smart Job Tracker

A modern full-stack MERN application that helps users manage and track their job applications while providing AI-powered resume analysis. The application enables users to organize job applications, monitor progress, upload resumes, and receive AI-generated insights to improve their chances of landing interviews.

---

## ✨ Features

- 🔐 Secure JWT Authentication (Login & Register)
- 💼 Add, Edit, Delete & Manage Job Applications
- 📊 Interactive Dashboard with Job Analytics
- 🤖 AI Resume Analyzer
- 📄 Resume PDF Upload & Text Extraction
- 🎯 Resume Match Score
- 💡 Missing Skills & AI Suggestions
- 🔍 Search & Filter Applications
- 📱 Fully Responsive UI
- 🎨 Modern SaaS-inspired Design

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/HirshanWarsi/smart-job-tracker.git
```

### Navigate into the project

```bash
cd smart-job-tracker
```

### Install Frontend Dependencies

```bash
cd job-tracker-frontend
npm install
```

### Install Backend Dependencies

```bash
cd ../job-tracker-backend
npm install
```

---

## ⚙️ Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

OPENAI_API_KEY=your_openai_api_key
```

---

## ▶️ Run the Project

### Backend

```bash
npm run dev
```

### Frontend

```bash
npm run dev
```
