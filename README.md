# 🚀 Smart Job Tracker

A full-stack MERN application that helps users track job applications and analyze resumes using AI.

## ✨ Features

- 🔐 JWT Authentication
- 💼 Job Application Management
- 📊 Dashboard Analytics
- 🤖 AI Resume Analyzer
- 📄 Resume PDF Upload
- 🔍 Search & Filter Jobs
- ✏️ Edit & Delete Applications
- 📱 Fully Responsive UI

## 🛠 Tech Stack

- React
- Node.js
- Express.js
- MongoDB
- Tailwind CSS
- JWT
- Multer
- PDF Parser
- OpenAI API (Mock + Real)

## 🚀 Getting Started

```bash
npm install
npm run dev
